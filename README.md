# Controls
<li>W/A/S/D - Move</li>
<li>Left Mouse Button - Shoot</li>
<li>R - Reload</li>
<li>ESC - Quit</li>

# Features
<li>A main menu</li>
<li>A working health bar that changes colour depending on the amount of health</li>
<li>Two different weapons</li>
<li>An ammo counter for each weapon</li>
<li>Different weapon effects for each weapon</li>
<li>A barely functioning inventory system</li>
<li>Four moving targets to shoot</li>
<li>A health potion that increases your health for a certain time period</li>
<li>A jump potion that increases your jump height for a certain time period</li>
<li>A speed potion that increases your speed for a certain time limit period</li>
<li>A locked door puzzle</li>
<li>Reloading ammo system</li>
